About This Demo

This project showcases the Google ‚ÄúAgent2Agent‚Äù (A2A) framework integerated with Angular UI: a master Assistant routes your queries to specialized agents (calculator, weather, GraphQL/data), streams their replies and artifacts (charts, files, JSON), and renders them here.

Available Agents

  • Assistant Agent: Dynamic intent classification + delegation & direct Query answering.
  • Calculator Agent: Computes math expressions.
  • Weather Agent: Fetches real-time weather by city.
  • GraphQL Tool Agent: Queries compressor mock data, builds charts & exports CSV.

Key Takeaways

  • Building a streamed AI UI with the A2A JS SDK.
  • Handling partial responses & rich artifacts (charts, downloads).
  • Clean, responsive design powered by AngularJS 1.x.
